옷에 껌이 붙은 경우 헤어스프레이나 얼음, 물, 파스를 충분히 뿌려 껌을 굳게 하고 그것을 긁어낸다.
옷에 립스틱이나 화장품이 묻은경우 알코올을 활용해 닦아 얼룩을 제거한다.
기름이 튀겼을 경우 주방세제로 먼저 비벼준 후 미온수로 세척한다.
혈흔이 묻었을경우 찬물로 빨아준다.
면소재의 이불은 액체 중성세제, 구스 이불은 미온수에 중성세제를 이용한다.
기능성 의류는 세탁기에 돌리지 않고, 수영복은 찬물에 손빨래 한다.
솜이나 깃털로 속을 채운 침구류의 경우 세탁 후 세제없이 한번 더 돌려야한다.
누렇게 된 양말은 레몬껍질 1개를 대야에 넣고 약/중불에서 10분간 삶아준다.
캔버스화는 3분간 물에 불려주고 칫솔로 밑창의 더러운 부분을 닦은뒤 헹군다.
패딩은 중성세제를 이용한 물세탁을 하면 가장 오래 입을 수 있다.
물이 빠지는 옷의 경우 소금물에 30분정도 담가 두었다가 식초를 한큰술 넣어주면 방지할 수 있다.
